Challenge

Train the best model you can with 50M parameters or less.

Simple Rules

  1. Model must be 50M parameters or fewer.
  2. Publish each run inside this repo under records/ so results are auditable.
  3. Each run folder should include a README.md, submission.json, and train.log.
  4. Use the Main Challenge track for the 10-minute budget runs.
  5. Use the Unlimited track for no-budget cap runs.
  6. If anything on this site conflicts with GitHub files, GitHub wins.

Setup and Get Started

Local setup:

git clone https://github.com/openai/N-challenge.git
cd N-challenge
pip install -r requirements.txt
python3 data/cached_fineweb10B.py 9

Main challenge launch (8x profile):

torchrun --standalone --nproc_per_node=8 train_gpt.py --config configs/train_gpt_8xh100.py

1x H100 fallback launch:

USE_FLASH_ATTN=0 torchrun --standalone --nproc_per_node=1 train_gpt.py --config configs/train_gpt_1xh100.py

Remote GPU note from project instructions: run heavy training/eval on a remote box through brix.

brix list
brix sync
# then run training on the box
