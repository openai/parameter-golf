## Novel levers surfaced

### MLPClip12 (serialization) — PR #1769 ★ ACTIONABLE ★

**Mechanism:** GPTQ int6 calibration uses a per-row outlier clip threshold (σ-clip). Our #1736
baseline inherited `MLP_CLIP_SIGMAS=10.0` — calibrated when the MLP was narrower. After the
stack widened to 4×MLP, the 10σ clip is too aggressive: it clips valid tail mass in wide MLP
rows, degrading quantization quality. Raising to 12σ admits those columns, trading slightly
coarser absolute resolution for better preservation of high-magnitude weights that carry
disproportionate MLP signal. The GPTQ calibration happens post-training; TTT runs on the
quantized artifact afterwards — so TTT cannot absorb this improvement.

**Δ-bucket:** ~−0.001 bpb (empirically −0.00096 over 7 seeds).

**Feasibility:** Trivial. One env-var. No code change. Zero risk.

**Next step:** Confirm our `records/track_10min_16mb/.../train_gpt.py` still has default 10.0,
then add `MLP_CLIP_SIGMAS=12` to run invocation. This is a free improvement — should be
absorbed into our baseline immediately (or specced as a 1-liner variant of spec 026 if spec 026
is still running).

### V-Gate (architecture) — PR #1770 (watch)

**Mechanism:** A single learned per-head scalar gates both (a) what information enters the V
projection and (b) how much that head's output contributes to the residual stream. This is
strictly different from AttnOutGate (our baseline), which gates the post-SDPA output. V-Gate
intercepts earlier in the attention pipeline. README claims "significantly improved model
performance" on the #1493 base stack.

**Δ-bucket:** Unknown without an ablation on #1736 base. 1.0796 result on non-CaseOps/
non-QuantGate stack is 0.013 above our floor — plausible the delta shrinks substantially on a
stronger base. Worth keeping on watch.

**Feasibility:** Small code change (add gating weight per head in attention module).

**TTT absorption:** No — this is a forward-pass architectural change.

**Next step:** Write idea file; re-evaluate if someone posts it on a #1736 base with competitive
numbers.
