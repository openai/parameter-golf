**What is QDMN?**  
This is my original idea of **Quad-Directional Modular Networks** — treating the model as a programmable fabric with four semantic directions (U/D/B) and explicit routes through a shared CoreNode.  
This is a creative/non-record submission focused on the architecture itself rather than chasing the lowest possible BPB.
