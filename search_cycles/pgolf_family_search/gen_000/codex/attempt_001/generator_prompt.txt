# Agent Prompt for Automatic Search Design

This file turns the search reset into something you can hand to a language model or coding agent.

Use it when you want the model to:

- generate hypothesis families automatically
- compile those families into minimal executable probes
- design a tournament
- identify which primitives need to be implemented

---

## How to use

Before sending the prompt, fill in:

- `<BASE_DESCRIPTION>`
- `<AVAILABLE_PRIMITIVES>`
- `<DISALLOWED_CHANGES>`
- `<CURRENT_FAILURE_PATTERNS>`
- `<BUDGET>`

Then ask the model to return **JSON only**.

```text
You are designing a bounded program search harness for optimizing training code under a fixed budget.

Your job is NOT to produce random patch ideas.

Given one frozen executable base, produce:
```
